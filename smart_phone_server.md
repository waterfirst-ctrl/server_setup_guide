# 📱 삼성 갤럭시 노트10 스마트폰 서버 구축 및 트러블슈팅 가이드

본 문서는 삼성 갤럭시 노트10(안드로이드 OS)을 활용하여 정적 HTML 웹 페이지 배포 및 학습용 미니 서버를 구축하는 전 과정과 실전 운영 중 발생한 트러블슈팅 내역을 정리한 품질팀 실전 가이드라인입니다.

---

## 1. 하드웨어 및 OS 공장초기화 (서버 최적화)
안정적인 백그라운드 리소스 확보를 위해 단말기를 완전히 제로 상태로 만드는 단계입니다.

* **기기 전체 초기화:** `설정 -> 일반 -> 초기화 -> 기기 전체 초기화` 진행
* **초기 세팅 필수 행동:**
  * **구글 및 삼성 계정 로그인 건너뛰기:** 백그라운드 동기화 및 자동 업데이트로 인한 리소스 낭비 방지
  * **화면 잠금 해제:** `설정 -> 잠금화면 -> 화면 잠금 방식 -> 설정 안 함` (재부팅 시 프로세스 차단 방지)

## 2. 서버 상시 구동을 위한 환경 설정 (안전장치)
안드로이드의 OS 차원의 백그라운드 프로세스 강제 종료(Sleep Mode)를 방지합니다.

* **개발자 옵션 활성화 및 화면 켜짐 유지:**
  1. `설정 -> 휴대전화 정보 -> 소프트웨어 정보 -> 빌드번호` 연속 7번 터치
  2. `설정 -> 개발자 옵션 -> 충전 중 화면 켜짐 상태 유지` 활성화 (상시 전원 공급 시 잠자기 진입 방지)
* **배터리 절전 예외 설정:**
  1. `설정 -> 배터리 및 디바이스 케어 -> 배터리 -> 앱별 배터리 사용 관리`에서 자동 절전 기능 일체 비활성화
  2. Termux 앱 속성 정보에서 **배터리 제한 없음(Unrestricted)**으로 설정
* **하드웨어 보호:** 상시 충전 시 배터리 팽창(스웰링) 및 발열 위험이 있으므로 `배터리 보호(85% 제한)` 기능을 켜고 통풍이 잘되는 곳에 배치합니다.

## 3. 네트워크 고정 IP 설정
공유기의 DHCP 스케줄에 의해 IP가 변경되어 접속이 끊기는 것을 방지합니다.

1. `설정 -> 연결 -> Wi-Fi` 이동 후 연결된 Wi-Fi 옆 ⚙️(톱니바퀴) 터치
2. **개인정보 보호:** `임의 MAC` ➡️ `휴대전화 MAC`으로 변경
3. **IP 설정:** `DHCP` ➡️ `고정(Static)`으로 변경
4. 현재 할당받은 단말기 IP 주소(예: `192.168.0.43`)를 확인 후 저장
