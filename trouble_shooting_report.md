스마트폰(Termux) 인프라 구축 과정에서 겪은 **‘파이썬 최신 버전 호환성 갈등’ 트러블슈팅 경험**과 ‘프론트-백엔드 분리(하이브리드) 아키텍처’는 실무에서도 매우 가치 있게 다뤄지는 엔지니어링 패턴입니다.

이번 트러블슈팅의 핵심 교훈을 정리하고, 이 가볍고 강력한 내장 모듈 서버 인프라를 활용해 앞으로 확장할 수 있는 **실무적이고 유용한 프로젝트 아이디어들**을 제안해 드립니다.

## 1. 이번 트러블슈팅의 핵심 요약 (엔지니어링 자산)

| 문제 상황 | 원인 | 최종 해결 전략 (품질 표준) |
| --- | --- | --- |
| **`pip install` 도중 멈춤 및 Rust (`maturin`) 에러** | 최신 Python 3.14 환경에서 모바일(ARM64)용 미리 빌드된 패키지(`.whl`)가 없어 기기 내 자체 컴파일 시도 시 자원 부족으로 뻗음. | **의존성 0%화 (하향 안정화)**<br>

## 2. 이 스마트폰 서버로 할 수 있는 의미 있는 프로젝트 TOP 4

스마트폰 서버의 가장 큰 무기는 "24시간 항상 켜져 있고, 전력 소모가 극도로 적으며, Wi-Fi로 상시 연결되어 있다"는 점입니다. PC 브라우저(Three.js/HTML)를 프론트엔드로 두고, 폰을 백엔드로 쓰는 하이브리드 구조를 확장하면 다음과 같은 것들을 만들 수 있습니다.

### ① 24시간 원격 파일 서버 & 웹 대시보드 (자체 클라우드)

* **개념:** 스마트폰의 대용량 스토리지(UFS 메모리)를 활용해 개인 파일 저장소를 만듭니다.
* **구현:** 폰 서버가 특정 폴더의 파일 목록을 JSON으로 리턴하면, PC 브라우저(Three.js) 화면에서 파일들이 3D 상자나 홀로그램 형태로 둥둥 떠다니는 리스트를 만듭니다. 클릭하면 PC로 즉시 다운로드되거나 텍스트를 읽어올 수 있습니다.
* **가치:** 외부 상용 클라우드(구글 드라이브 등)를 쓰지 않고 고속 로컬 네트워크 안에서 나만의 보안 클라우드를 운영할 수 있습니다.

### ② 모바일 센서 데이터 수집기 (IoT Edge 장치)

* **개념:** 노트10 내부에 있는 실제 하드웨어 센서(배터리 전압, 가속도 센서, 조도 센서 등) 정보를 수집하는 API 서버로 진화시킵니다.
* **구현:** Termux의 `termux-battery-status` 또는 센서 API 명령어를 파이썬의 `subprocess` 모듈로 호출하여 실시간으로 긁어옵니다. PC의 Three.js 화면에서는 스마트폰 모양의 3D 모델이 폰의 실제 배터리 잔량이나 충전 상태에 따라 실시간으로 변하는 모니터링 룸을 구축합니다.
* **가치:** 안드로이드 공기계를 고가의 산업용 IoT 수집 장비(Edge Device)로 재탄생시키는 효과를 냅니다.

### ③ 라이트한 웹 스크래퍼 & 알림 봇 (자동화 백엔드)

* **개념:** 파이썬 내장 라이브러리(`urllib.request`)를 활용해 주기적으로 특정 기술 사이트의 공지사항, 환율, 혹은 특정 주가 데이터를 긁어오는 스크래퍼를 폰 내부에서 백그라운드로 상시 구동합니다.
* **구현:** 데이터가 갱신되거나 특정 조건(예: 목표가 도달)을 만족하면 폰 서버가 텔레그램 보스(Bot) API를 호출해 내 스마트폰으로 알림을 톡으로 쏘아주거나, PC 3D 대시보드의 알림 게이지를 붉은색으로 점등시킵니다.
* **가치:** 무거운 PC를 밤새 켜두지 않아도 스마트폰이 알아서 새벽 내내 초절전 상태로 자동화 스케줄을 처리합니다.

### ④ 임시 개발/테스트용 로컬 REST API 샌드박스

* **개념:** 데이터 분석이나 프론트엔드 시각화 툴(Three.js, React 등)을 개발할 때, 가상의 더미 데이터(Mock Data)를 뱉어내는 훌륭한 로컬 테스트 베드로 활용합니다.
* **구현:** 데이터베이스(DB) 패키지를 깔 필요 없이, 파이썬 내장 `sqlite3` 모듈(가장 가볍고 컴파일이 필요 없는 내장 파일 DB)을 연동해 가벼운 CRUD(등록/조회/수정/삭제) 스토리지 서버를 구성합니다.
* **가치:** 메인 개발 PC의 개발 환경을 더럽히지 않고, 완전하게 격리된 하드웨어 환경에서 통신 테스트를 완벽하게 수행할 수 있습니다.
